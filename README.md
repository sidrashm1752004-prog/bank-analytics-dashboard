
# Bank Analytics & Financial Insights Project

## Project Overview  
This project focuses on end-to-end analysis of banking data to uncover insights related to loan performance, customer transactions, and financial trends.  

Two datasets were used to perform analysis across Excel, Tableau, Power BI, and SQL. The project demonstrates how raw data is transformed into interactive dashboards and business insights.

## Objectives  
- Analyze loan distribution and recovery performance  
- Understand customer transaction behavior (Debit vs Credit)  
- Identify high-value customers  
- Track branch-wise performance and monthly growth  


## Tools & Technologies  
- Excel  
- Tableau  
- Power BI  
- SQL  


## Project Structure  
- **Data** → Banking datasets  
- **Dashboards** → Power BI & Tableau files  
- **SQL** → Queries and analysis  
- **Presentation** → Project PPT

### Transaction Dataset

- Total Debit → ₹127.29M

## SQL Highlights  
- Window Functions (DENSE_RANK, LAG)  
- CTEs and Views  
- Stored Procedures  
- KPI calculations  


##  Dashboards  
- Excel Dashboard → Loan analysis  
- Tableau Dashboard → Loan trends & categories  
- Power BI Dashboard → Transaction insights  



## Key Insights  
- High-value customers drive major revenue  
- Loan recovery is strong compared to disbursed amount  
- Some branches show high debit dependency  
- Monthly trends reveal seasonal patterns  



##  How to Use
